Glossary

Div tag
a division or a section in an HTML document.
Body tag
defines the document's body.
footer tag
a footer for a document or section.
Header tag
a container for introductory content or a set of navigational links.
Hr tag
a thematic break in an HTML page
Meta tag
metadata about an HTML document. Metadata is data (information) about data.
